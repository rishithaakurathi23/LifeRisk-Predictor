FutureFit – AI-Powered Future Disease Risk Predictor Tagline: Predict. Prevent. Stay Healthy. Table of Contents 1.Project Overview 2.Features 3.Dataset 4.Machine Learning Approach Contributing License Project Overview FutureFit is an AI-powered tool designed to predict an individual’s future risk of lifestyle-related diseases such as diabetes, obesity, and heart disease. By analyzing daily lifestyle and dietary patterns, the system provides: Predictive risk scoring (High Risk / Low Risk) Personalized health advice for special conditions like diabetes, pregnancy, heart disease, or cancer This empowers users to make informed lifestyle decisions and take preventive measures. Features Predictive ML Model: Random Forest Classifier predicts disease risk based on input features. User-Friendly Input: Handles fuzzy/misspelled meal entries. Personalized Recommendations: Provides diet advice for special health conditions. Visual Insights: Feature importance plotted to show major risk contributors. Dataset Type: Simulated dataset with 1000 samples Features: Age (numerical) Gender (categorical) Breakfast, Lunch, Dinner, Snacks (categorical: junk, normal, vegetables, fruits) Target: Disease_Risk (0 = Low, 1 = High) Machine Learning Approach Algorithm: Random Forest Classifier (ensemble learning) Preprocessing: One-Hot Encoding for categorical variables Numerical features passed as-is Pipeline: Data preprocessing + model training integrated into a single pipeline for efficiency
